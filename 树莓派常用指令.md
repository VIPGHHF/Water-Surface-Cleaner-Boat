# 树莓派 (Ubuntu) 极客高频指令速查表

## 1. 系统与电源控制 (掌控生死)
* `sudo shutdown -h now` : [安全关机] 拔电源前必点，防止 SD 卡文件系统损坏。
* `sudo reboot` : [重启系统] 修改底层配置（如 config.txt）后使其生效。
* `vcgencmd measure_temp` : [查看温度] 实时查看树莓派 CPU 温度，防止过热降频。
* `htop` : [任务管理器] 实时监控 CPU 核心占用和内存消耗（按 `q` 退出）。如果没有需执行 `sudo apt install htop` 安装。

## 2. 硬件与外设排查 (X光透视)
* `lsusb` : [查看 USB] 列出所有已连接的物理 USB 设备。
* `ls -l /dev/video*` : [查摄像头] 确认 CSI 摄像头是否成功挂载（寻找 video0）。
* `ls /dev/ttyA*` 或 `ls /dev/serial*` : [查串口] 检查与单片机通信的物理串口是否开启。
* `sudo dmesg -w` : [内核日志] 实时监听底层硬件拔插反馈和报错（如欠压警告）。按 `Ctrl + C` 退出。
* `pinout` : [引脚图鉴] 在终端直接绘制出树莓派 40 针 GPIO 的物理排线图。

## 3. 文件与目录穿梭 (外科手术刀)
* `pwd` : [当前路径] 打印当前所在的绝对路径（防迷路）。
* `cd <文件夹名>` : [进入目录] 进入指定文件夹（如 `cd Water-Surface-Cleaner-Boat`）。
* `cd ..` : [返回上级] 退回到上一级目录。
* `ls -la` : [详细列表] 列出当前目录下的所有文件，包括隐藏文件和读写权限。
* `cat <文件名>` : [快速预览] 直接在终端打印出文件内容，不进入编辑模式。
* `sudo nano <文件名>` : [修改文件] 打开终端编辑器。修改后按 `Ctrl + O` -> `Enter` 保存，按 `Ctrl + X` 退出。

## 4. 进程与死机急救 (暴力狙击)
* `Ctrl + C` : [紧急刹车] 强行中断终端里正在运行的当前程序。
* `ps aux | grep python` : [雷达扫描] 查找所有后台偷偷运行的 Python 进程，并获取其进程号 (PID)。
* `kill -9 <PID>` : [一击毙命] 填入上述查到的进程号，强行物理超度卡死的程序，释放硬件占用。

## 5. 网络与权限
* `sudo <命令>` : [尚方宝剑] 获取超级管理员权限强行执行命令。
* `ifconfig` 或 `ip a` : [查看 IP] 检查树莓派当前的局域网 IP 地址。
* `ping baidu.com` : [测试网络] 测试外网连通性与延迟。按 `Ctrl + C` 退出。
